Discord Game Tracker

A small discord bot I built for a server I share with some friends. It primarily tracks games we'd like to play and preforms a handful of other functions. This bot was never meant as a terribly serious project, but as an excuse to check out the discord and steam APIs.

This project is hosted solely on my local machine, source code is visible on github.

Zoho CRM API

This project is a simple python wrapper for the Zoho CRM web API. It has since been rendered obsolete by the release of a V2 for their API as well as an official python wrapper. This project uses simple token authentication and support all basic CRUD operations as well as a handful or more advanced features for handling complex CRM objects.

This project is available for download on pypi but has not been updated or maintained since early 2019 as it is completely depreciated at this point.

Craigslist Scraper

Another simple project one could hypothetically use to efficiently monitor craigslist for newly posted deals. This tool support a variety of different user-defined queries and stores everything in a sqlite database queried with sqlalchemy. One could theoretically define and monitor a specific bounding box for new apartment listings or used car sales within a certain price range.

Source code is visible on github.
