Discord Game Tracker

A small discord bot I built for a server I share with some friends. It tracks games we'd like to play, has a few management commands, and allows us to play music in a voice channel. This bot was built to add some fun functionality to a private server and was never taken too seriously. It was however, an excellent opportunity to practice asynchronous programming in python.

This project is simply hosted on my personal, local machine. Some source code is visible on github. The active branch is in a private repository.
